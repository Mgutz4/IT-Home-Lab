# IT Home Lab Portfolio

This repository showcases hands-on IT labs designed to simulate real-world system administration and help desk scenarios. The labs focus on Active Directory management, Group Policy configuration, user support workflows, and network traffic analysis.

---

## 🔧 Labs Included

### 🖥️ Active Directory & DNS Lab
- Deployed and configured a Domain Controller
- Set up DNS for internal name resolution
- Joined client machines to the domain
- Verified authentication and connectivity
- Troubleshot DNS and domain-related issues

---

### 🔐 Group Policy (Workstation Control)
- Created and linked GPOs to Organizational Units (OUs)
- Restricted Control Panel access for Employees OU
- Applied user-based policies using Administrative Templates
- Verified policy application using `gpupdate` and `gpresult`
- Demonstrated centralized management of user environments

---

### 🔑 Password Reset & Account Unlock
- Simulated user account lockouts due to failed login attempts
- Reset passwords and unlocked accounts using ADUC
- Verified login access on domain-joined client machines
- Analyzed security logs in Event Viewer (Event ID 4740, 4625)
- Demonstrated real-world help desk support workflow

---

### 🌐 Wireshark Traffic Analysis
- Captured live network traffic using Wireshark
- Filtered and analyzed DNS, TCP, and ICMP traffic
- Identified communication between client and external servers
- Observed packet structure and protocol behavior
- Demonstrated basic network troubleshooting techniques

---

## 🛠️ Technologies Used

- Active Directory (AD DS)
- Group Policy (GPO)
- Windows Server 2022
- Windows 10
- Wireshark
- Command Line Tools (`ping`, `nslookup`, `gpupdate`, `gpresult`, `whoami`)
- Event Viewer

---

## 🧠 Skills Demonstrated

- User Account Management
- Group Policy Administration
- Troubleshooting Authentication Issues
- Network Traffic Analysis
- DNS Configuration & Troubleshooting
- Help Desk Support Workflows
- System Administration Fundamentals

---

## 🎯 Purpose

This lab environment was built to develop practical, job-ready IT skills by simulating real-world enterprise scenarios. Each lab reflects tasks commonly performed in Tier 1 / Tier 2 IT support roles.
